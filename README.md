# 区块链原理模拟器

## 项目概述
本项目通过模拟实现一个简化版的工作量证明区块链，旨在揭示区块链技术的核心运行机制，包括区块结构、哈希链接、工作量证明和链的完整性验证。这是一个命令行演示程序，不涉及网络通信。

## 核心功能
- 区块创建与哈希计算：能够根据数据创建新区块，并计算其加密哈希值。
- 工作量证明：实现简单的“挖矿”过程，通过调整Nonce找到满足特定难度的哈希。
- 区块链构建：将区块通过哈希值按时间顺序链接成链。
- 链完整性验证：提供验证功能，确保区块链未被篡改。

## 技术栈
- 编程语言：Python
- 核心库：hashlib(用于SHA-256哈希计算)，time, json
- Web框架：Flask

## 运行指南
### 命令行版本
运行模拟：`python main.py`

演示验证：程序将打印出整个链的信息，并调用验证功能。之后，尝试修改链中某个区块的数据，再次验证，观察验证失败。

### 网页版本
1. 安装依赖：`pip install flask`
2. 运行Web应用：`python app.py`
3. 在浏览器中访问 `http://127.0.0.1:5000/`

网页界面允许：
- 查看区块链
- 添加新区块
- 验证链完整性
- 演示篡改区块

## 文件结构
- `block.py`: 定义Block类
- `blockchain.py`: 定义Blockchain类
- `main.py`: 主程序演示（命令行）
- `app.py`: Flask Web应用
- `templates/index.html`: 网页模板
- `README.md`: 项目文档

## 总结与扩展
本模拟器清晰地展示了区块链不可篡改性和工作量证明的本质。未来可扩展为简单的P2P网络，允许节点间同步区块链，并实现基础的交易模型和余额系统。
